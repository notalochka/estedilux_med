# Швидкий старт

## Встановлення

1. Встановіть залежності:
```bash
npm install
```

## Запуск проекту

### Режим розробки
```bash
npm run dev
```

Відкрийте браузер за адресою: http://localhost:3000

### Перевірка типів TypeScript
```bash
npm run type-check
```

### Збірка для продакшену
```bash
npm run build
npm start
```

## Структура проекту

```
estedilux-med/
├── public/              # Статичні файли
│   └── locales/         # Файли перекладів (uk, en)
├── src/
│   ├── components/      # React компоненти
│   │   ├── Header/      # Шапка сайту
│   │   ├── Footer/      # Підвал сайту
│   │   └── Hero/        # Головна секція
│   ├── pages/           # Next.js сторінки
│   │   ├── index.tsx    # Головна сторінка
│   │   ├── courses/     # Сторінка курсів
│   │   ├── about/       # Про нас
│   │   └── contact/     # Контакти
│   ├── styles/          # Глобальні стилі
│   ├── types/           # TypeScript типи
│   └── config/          # Конфігурація
├── tsconfig.json        # TypeScript конфігурація
└── next.config.js       # Next.js конфігурація
```
